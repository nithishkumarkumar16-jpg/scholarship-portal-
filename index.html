Important Information

Post Matric Scholarships support students studying after Class 10. These scholarships help reduce financial burden for eligible students.

Eligible Categories

  • SC / ST Students
  • BC / MBC / DNC Students
  • Persons with Disabilities (PwD)

Scholarship Benefits

  • SC / ST: Full or partial tuition fee + maintenance allowance
  • BC / MBC / DNC: Fee concession / maintenance grant
üìå Amounts differ yearly. Check latest official notification.

Important Notes

  • Scholarship via Direct Benefit Transfer (DBT)
  • Aadhaar must be linked with bank account
  • Name and DOB must match in all documents
  • Apply through UMIS or NSP only

üèõ Apply via UMIS Portal

College-based application

Before Applying (Required)

  • UMIS User ID & Password from college
  • Aadhaar linked with bank account
  • 10th / 12th / previous marksheets
  • Income & community certificate
  • Bank passbook (with name)
  • Fee receipt / bonafide certificate

Steps to Apply

  1. Login to college UMIS portal
  2. Open "Scholarship / DBT Section"
  3. Select correct scheme (SC / ST / BC / MBC / DNC)
  4. Enter personal details (match Aadhaar exactly)
  5. Fill academic and course details
  6. Enter bank account details (Aadhaar linked)
  7. Upload required documents
  8. Verify all details carefully
  9. Submit the application
  10. Save or print acknowledgement

Important Notes

  • Name and DOB must match Aadhaar and bank
  • Incorrect details ‚Üí rejection
  • Track status through UMIS
  • Contact college coordinator if login fails

üåê National Scholarship Portal

NSP application

üåê Open NSP Portal

When to use NSP

  • If college does not provide UMIS login
  • If scheme requires national-level submission

Steps to Apply

  1. Click "New Registration"
  2. Accept guidelines and enter details
  3. Create login using OTP
  4. Login and apply for scholarship
  5. Select correct Post Matric scheme
  6. Fill personal, academic & bank details
  7. Upload required documents
  8. Submit and track application status

Important Tips

  • Use same name & DOB in Aadhaar, bank, certificates
  • Keep mobile number active for OTP
  • Check application status regularly
